# ![](assets/logo.png) Fancy Ice Cream

A Minecraft mod to add ice cream recipes.

![](assets/screenshot.png)

## Prerequisites

- [Minecraft Forge](https://files.minecraftforge.net/net/minecraftforge/forge/)
